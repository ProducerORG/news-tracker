Was macht der NewsTracker?

Er sammelt Meldungen von bestimmten Pressequellen. Die Tabellenansicht zeigt die aktuellsten Einträge in Echtzeit. Sortieren Sie durch Klick auf die Spalte.

Unten findet sich eine Liste der aktuell angezapften Quellen. Beantragen Sie die Einbindung einer neuen Quelle über das Formular. Wenn Sie bestehende Quellen deaktivieren, werden fortan keine neuen Beiträge mehr aus dieser Quelle ermittelt, bis sie diese wieder aktivieren.
